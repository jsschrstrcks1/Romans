# Habakkuk

Reference material, exegetical notes, and (potentially) book project for the prophet Habakkuk.

## Purpose

This folder houses research and writing related to the book of Habakkuk. It is the working library that supports:

- The 6-sermon series **Faith in the Dark** (Habakkuk 1-3), preached April-May 2026
- Future book project (TBD)
- Ongoing exegetical, theological, and historical study

The actual sermon manuscripts live at `/sermons/ot/habakkuk/`. This folder is for the research that feeds them.

## Structure

```
habakkuk/
├── README.md                  # This file
└── references/                # Source material
    ├── baker-encyclopedia-mccomiskey.md
    └── adrian-rogers-sermons.md
```

Additional subdirectories will be added as the work develops:
- `exegetical-notes/` — verse-by-verse working notes
- `word-studies/` — Hebrew word work
- `historical-context/` — background on Babylon, Jehoiakim, etc.
- `book/` — if and when a book project begins

All quotations and references in these files are preserved as received from primary sources or trusted secondary sources. Before any quotation is used in a sermon, it must be verified per the `careful-not-clever` skill standards.
