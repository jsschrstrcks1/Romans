# Quotes & Reference Materials

Source materials for quote verification and direct citation in sermon preparation and book manuscript work.

## Contents

### spurgeon/
Charles Haddon Spurgeon (1834–1892) sermon PDFs sourced from SpurgeonGems.org.

- `sermons/` — Individual sermon PDFs (chs1.pdf–chs3563.pdf)
- `sermons/indexes/` — Scripture and topical indexes

**Usage**: For quote verification and direct quotation. Always cite the sermon number and title when quoting. Cross-reference with the NT Scripture Index (sindex_nt.pdf) to find sermons on specific Romans passages.

### illustrations/
Reusable biographical and topical illustrations with sourcing, application notes, cross-reference passages, and usage logs.

- `cowper-william.md` — William Cowper's four suicide attempts, conversion, and "God Moves in a Mysterious Way" (Olney Hymns, 1779). Tags: depression, providence, redemption through suffering, hiddenness of grace.

**Usage**: Each entry includes a usage log. When you use an illustration in a new sermon, add a row to the log so it can be tracked across the corpus.
