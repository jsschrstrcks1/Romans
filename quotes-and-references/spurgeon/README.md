# Spurgeon Sermons

Charles Haddon Spurgeon (1834–1892) — full sermon collection sourced from [SpurgeonGems.org](https://www.spurgeongems.org/spurgeon-sermons/).

## Files

### sermons/
Individual sermon PDFs numbered `chs1.pdf` through `chs3563.pdf` — the complete Metropolitan Tabernacle Pulpit and New Park Street Pulpit.

Includes special files:
- Combined sermons (e.g., `chs7-8.pdf`, `chs39-40.pdf`)
- Variant editions (e.g., `chs388a.pdf`, `chs388b.pdf`, `chs364A.pdf`, `chs369A.pdf`)

### sermons/indexes/
| File | Contents |
|---|---|
| `chstix.pdf` | General Sermons Index |
| `sindex_nt.pdf` | New Testament Scripture Index |
| `sindex_ot.pdf` | Old Testament Scripture Index |
| `chsp1–chsp13.pdf` | Volume prefaces (Vols 1–13) |
| `chsofs.pdf` | "Our First Sermon" |
| `chsof7.pdf` | "Our First Seven Years" |

## Finding Sermons on Romans

Use `sindex_nt.pdf` — navigate to the Romans section for a complete list of sermons cross-referenced to specific passages.

## Re-downloading / Updating

```bash
cd quotes-and-references/spurgeon/
bash download-spurgeon.sh
```

The script is idempotent — it skips already-downloaded files and removes empty/failed downloads.
