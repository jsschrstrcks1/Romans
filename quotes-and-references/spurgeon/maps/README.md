# Spurgeon Sermon Maps

Reference indexes for locating and verifying Spurgeon quotes during sermon writing.

## Scope — Read This First

These maps index **Spurgeon's sermons only**. They are:

- **Separate** from `.claude/sermon-map.md` and `.claude/theological-map.md`, which index your own sermons
- **Not to be merged** into any broader sermon index system
- **Not a source for sermon content** — only for finding and verifying short quotes

Permitted uses:
- Finding a Spurgeon quote on a topic while writing
- Verifying the exact wording of an attributed Spurgeon quote
- Confirming which sermon a quote comes from before citing it

Not permitted:
- Reproducing full sermons or extended passages
- Treating Spurgeon's content as your own

> Always cite: *"Quote." — C.H. Spurgeon, Sermon Title, No. N, Date.*

## How to Use

**Finding a quote on a topic:**
1. Open `subject.md` or `theology.md`
2. Find the topic → note the sermon number(s)
3. Open `../sermons/chsN.md` and search for the quote

**Verifying an attributed quote:**
1. If you know the scripture text, use `scripture-nt.md` or `scripture-ot.md`
2. Find the passage → check the listed sermons
3. Open the sermon file and confirm the exact wording

**Citation format:**
> "Quote here." — C.H. Spurgeon, *Sermon Title*, No. N, Date
