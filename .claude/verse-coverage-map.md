# Verse Coverage Map

*Tracks preaching coverage verse by verse across the sermon corpus. Shows whether each passage has been preached as a primary text or appears as a secondary reference in another sermon.*

*Last updated: March 21, 2026*

---

## How to Read

| Marker | Meaning |
|--------|---------|
| **■ PRIMARY** | This passage was the main preaching text for at least one sermon |
| **◇ SECONDARY** | This passage appears as a cross-reference, supporting text, or illustration anchor in another sermon — but was never the primary text |
| **⬜ GAP** | No documented coverage — not preached, not referenced |
| **⚠️ PARTIAL** | Some verses in the range are covered; others are not |

**Column definitions:**
- **Range** — verse range, using natural sermon divisions (not individual verses)
- **Status** — ■ / ◇ / ⬜ / ⚠️
- **Primary Sermon(s)** — filename(s) where this range was the primary text (blank if status is ◇ or ⬜)
- **Secondary In** — filename(s) where this range appears as a cross-reference (abbreviated)
- **Date** — preaching date if primary (from date-map)

**Scope note:** This map currently covers Romans (the active series) comprehensively. Other books are added when the COVERAGE operation fires during indexing of non-Romans sermons. Secondary references are extracted from sermon-map.md Subject columns — not guaranteed exhaustive.

## Maintenance

This file is maintained by **Operation 10: COVERAGE** in the sermon-map skill.

- **When a new Romans sermon is indexed**: Add a row to the appropriate chapter table. Mark ■ PRIMARY with filename and date.
- **When a cross-reference is noted in a sermon-map entry**: Add a ◇ SECONDARY row if the passage doesn't already have one.
- **When a gap is filled**: Change ⬜ GAP → ■ PRIMARY. Update the Gap Summary table.
- **Non-Romans books**: Add a new `## [Book]` section following the same chapter/table format when sermons from that book are indexed.
