# Theological Taxonomy

*A tiered classification system for the full sermon corpus. Maps every doctrinal topic to its standard systematic-theology locus, sub-category, and specificity level. Designed for cross-referencing with `theological-map.md`, `sermon-map.md`, and `preaching-gap-analysis.md`.*

---

## How to Use This File

- **Tier 1** = Major locus (e.g., Theology Proper, Soteriology)
- **Tier 2** = Sub-doctrine (e.g., Divine Attributes, Justification)
- **Tier 3** = Specific teaching point (e.g., Aseity, Imputation of Righteousness)
- **Sermon refs** point to filenames in the corpus
- **Strength** = how thoroughly the corpus covers this point (Strong / Moderate / Light / Gap)
- **Tradition** = confessional or historical-theological stream
