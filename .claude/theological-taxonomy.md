# Theological Taxonomy

*A tiered classification system for all doctrinal content in the sermon corpus. Each category maps to the theological-map.md entries and sermon-map.md file references. Use this to locate, classify, and cross-reference any theological topic across 200+ sermons.*

---

## How to Use This File

- **Tier 1** = Major locus (e.g., Theology Proper)
- **Tier 2** = Doctrine within that locus (e.g., Divine Attributes)
- **Tier 3** = Specific teaching (e.g., Immutability)
- **Strength** = How heavily the corpus treats this topic (Very Strong / Strong / Moderate / Light / Gap)
- **Key Files** = Primary sermon files where this doctrine is taught
- **Tradition** = Confessional/theological tradition the teaching aligns with
